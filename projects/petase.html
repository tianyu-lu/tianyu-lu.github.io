Abstract

To optimize a proteinâ€™s catalytic activity, we train two neural networks. One, a generative model that sees a dataset of protein sequences and generates new sequences based on those previously seen. For this we use a recurrent neural network (RNN). Two, we need a discriminative model that for a given protein sequence, is able to accurately predict its catalytic activity. For this, we train a variety of machine learning models and choose the best performing model. In addition, for the discriminative model, we apply transfer learning to improve model performance on small datasets such as that for PETase.
